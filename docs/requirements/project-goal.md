# Цель проекта (по SMART)

Создать цифровую платформу взаимного обучения для пользователей университетов и частных лиц, в которой пользователи обмениваются практическими или теоретическими навыками в формате «равный равному» через синхронные видео-комнаты, с последующей верификацией качества передачи знаний через механизм взаимной оценки (рейтинг «+/–»).

# Миссия проекта

Сделать обмен знаниями бесплатным и доступным для каждого. Мы создаём платформу, где люди могут учить друг друга без денег — один делится тем, что умеет, другой помогает в ответ. Университеты получают готовый инструмент для развития студентов и преподавателей, а обычные пользователи — возможность бесплатно осваивать новые навыки. Наша главная идея: у каждого есть чему научить, и у каждого есть чему научиться.

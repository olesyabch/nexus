# MVP (Minimum Viable Product)

## Что входит в MVP

- Регистрация и аутентификация
- Профиль с двумя списками: «Чему могу научить» / «Чему хочу научиться» (для университетских пользователей — факультет, курс)
- Простой поиск по категориям навыков, факультету, курсу
- Видео-комнаты через Zoom
- Кнопки оценки «+» и «–» после сессии
- Базовая система скилл коинс
- Административная панель для университета
- Разделение пользователей на «университетских» (полный доступ) и «частных» (с ограничениями)
- Ограничения для частных пользователей: не более 5 уроков в месяц, базовый поиск

## Что исключается из MVP

- Сложный алгоритм рекомендаций (на MVP — простой поиск)
- Мобильные приложения (только адаптивный веб)
- Встроенные видео-комнаты и записи уроков
- Сложная модерация (на MVP — ручная)
- Интеграция с университетскими информационными системами

# Паспорт проекта Nexus

## Смета запланированных расходов на разработку MVP

### Общая концепция финансового планирования

Проект коммерческой платформы взаимного обучения Nexus разработан силами внутренней проектной команды без привлечения внешних инвестиций на этапе создания MVP (минимально жизнеспособного продукта). Экономическая оценка проекта строится на основе капитализации интеллектуального труда команды.

Расчёт бюджета MVP выполнен путём сложения трудозатрат участников (в человеко-часах) согласно их проектным ролям и средним рыночным ставкам для Junior-специалистов в ИТ-отрасли.

Архитектура системы и конвейер разработки были сознательно спроектированы с использованием бесплатных тарифных планов (Free-tier) современных облачных платформ и ИИ-инструментов, что позволило зафиксировать материальные операционные затраты на этапе прототипирования на отметке 0 рублей.

# Разложение задач по методике MoSCoW

## Must have (обязательно для MVP)

1. Регистрация и вход через почту
2. Профиль пользователя с указанием факультета, курса, группы (для университетских пользователей)
3. Добавление навыков («могу» и «хочу»)
4. Поиск других пользователей по навыкам с фильтром по факультету и курсу
5. Просмотр чужого профиля
6. Чат
7. Кнопки оценки «+» и «–» после урока
8. Отображение рейтинга в профиле
9. Базовая система Skill Coins
10. Автоматическое напоминание об уроке (уведомление)
11. Административная панель для университета (просмотр активности пользователей, количество проведённых уроков, выгрузка отчётов)
14. Роли пользователей: университетский пользователь, частный пользователь, администратор университета
15. Разделение пользователей на «университетских» (полный доступ) и «частных» (с ограничениями)
16. Ограничения для частных пользователей (3 приветственных Skill Coins)
17. Покупка Premium-подписки и разовых покупок Skill Coins

## Should have (очень важно, но можно перенести на следующий спринт)

1. Вход через соцсети (Google, VK ID) как дополнительный способ
2. Приветственный опрос при регистрации
3. Сортировка результатов поиска по рейтингу
4. Чат во время урока
5. Аватар пользователя
6. Уведомление о новом менторе
7. Штраф за пропуск урока (снижение рейтинга)
8. Поощрение бонусными скилл коинс за текстовые отзывы
9. Онлайн-статус в поиске
10. Фильтр поиска по преподавателям / студентам
11. Премиум-подписка для частных пользователей (безлимит уроков, приоритет в поиске)

## Could have (хорошо бы, но можно отложить на версию 2.0)

1. Предпросмотр урока (демо-видео ментора)
2. Запись урока (с согласия обеих сторон)
3. Покупка  за деньги
4. Ежемесячный рейтинг лучших (топ-10 наставников) с передачей данных в деканат для поощрения
5. Форум или сообщество
6. История успеха реального пользователя на главной странице
7. Таймер урока
8. Дефицит скилл коинс с предложением провести урок

## Won't have (не делаем в текущей версии)

1. Мобильное приложение (iOS и Android)
2. Алгоритмический matching на основе AI
3. Интеграция с внешними календарями (Google Calendar, Outlook)
4. Система достижений и бейджей
5. Интеграция с YouTube для демо-уроков

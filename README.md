# Nexus - платформа обмена навыками

## О проекте
Цифровая платформа для обмена навыками между пользователями университетов и частными лицами в формате «равный равному» через синхронные видео-комнаты с системай взаимной оценки.

## Цель проекта (по SMART)
Создать цифровую платформу взаимного обучения для пользователей университетов и частных лиц, в которой пользователи обмениваются практическими или теоретическими навыками в формате «равный равному» через синхронные видео-комнаты, с последующей верификацией качества передачи знаний через механизм взаимной оценки (рейтинг «+/–»).

## Миссия
Сделать обмен знаниями бесплатным и доступным для каждого. У каждого есть чему научить, и у каждого есть чему научиться.

# ДОРОЖНАЯ КАРТА (ROADMAP) РАЗВИТИЯ ПРОДУКТА NEXUS

## Этап 1. Разработка и тестирование MVP (Текущий статус)
* **Реализовано в рамках MVP:**
  * Сборка адаптивного веб-интерфейса через ИИ Qwen на основе Hi-Fi макетов из Figma.
  * Интеграция модуля аутентификации и онбординга (Supabase Auth) с трехуровневым ветвлением (Частные/ВУЗ → Админ/Участник → Отлепление преподавателя на шаге выбора группы).
  * Базовая база данных на PostgreSQL (таблицы `profiles`, `universities`, `institutes`, `groups`, `skills_list`, `lessons`).
  * Базовый баланс Skill Coins (логика списания, начисления и заморозки валюты на стороне бэкенда Supabase).
  * Встроенный модуль текстовых P2P-чатов для координации встреч и обмена внешними ссылками.
  * Панель администратора с настроенными политиками безопасности Row Level Security (RLS) для сквозного мониторинга проведенных занятий.
* **Исключено из MVP:** Мобильное приложение, встроенные видеокомнаты, автоматическая запись уроков, интеграция с вузовскими LMS, сложный рекомендательный ИИ-подбор менторов.

---

## Этап 2. Пилотный запуск и стабилизация
* Развертывание разработанного MVP на тестовую аудиторию одного пилотного Института внутри университета.
* Сбор обратной связи через встроенные формы фидбека и проведение UX-интервью со студентами и преподавателями.
* Мониторинг нагрузки на базу данных PostgreSQL в Supabase, оптимизация индексов и устранение критических багов.

---

## Этап 3. Масштабирование и интеграция в экосистему ВУЗа
* Переход с бесплатного тарифа Free-tier в Supabase на коммерческий тариф Pro Plan для расширения объемов БД и поддержки высокой посещаемости.
* Разработка модуля генерации автоматических отчетов (Топ-10 наставников института) с интеграцией API для передачи данных в деканат для академического поощрения.
* Внедрение полноценной системы рейтингов, оценок и текстовых отзывов после каждого урока.
* Запуск внутренней новостной ленты и комьюнити-форума для обсуждения образовательных трендов внутри университета.

---

## Этап 4. Интеллектуальный мэтчинг и мобильная экосистема
* Внедрение рекомендательного ИИ-сервиса (Matching Service) для автоматического подбора менторов на основе истории обучения, курса и успеваемости студента.
* Интеграция с внешними календарями (Google Calendar, Яндекс.Календарь) через API для автоматического планирования расписания.
* Внедрение геймификации: система цифровых бейджей, достижений за наставничество и организация групповых бартерных лекций.
* Адаптация бэкенда Supabase и разработка официального мобильного приложения под iOS и Android.

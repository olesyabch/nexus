# Проблемы, которые решает проект

## 1. Финансовый барьер
Качественное индивидуальное обучение (репетиторы, тренеры) стоит дорого. Согласно статистике, до 40% пользователей не могут позволить себе нанять специалиста.

## 2. Инфляция контента
На платформах вроде YouTube и Skillbox переизбыток записанных лекций. У пользователя падает вовлеченность из-за отсутствия живого контакта и обратной связи.

## 3. Асимметрия навыков
Один пользователь может быть гениальным программистом, но слаб в иностранном языке. Отсутствует инфраструктура для бартерного обмена знаниями.

## 4. Проблема доверия и результата
Существующие платформы для поиска напарников не гарантируют качество. Механизм оценки «+» и «–» решает проблему недобросовестности.
